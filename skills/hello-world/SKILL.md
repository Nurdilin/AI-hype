Use the hello world skill to greet users with a friendly message and some useful system information.
This skill can be triggered when they enter the phrase "hello world" or similar greetings and it will respond with "Hello!

1. run the following script to get system information, the os, the kernel version, the date and the uptime:
[script](./scripts/get-info)

2. respond with [template](./TEMPLATE.md) and fill in the system information obtained from the script.
